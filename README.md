# 🗂️ Sorter — цифровой дворецкий для ваших файлов

**Sorter** — это легковесный кроссплатформенный инструмент для автоматической сортировки и очистки файлов. Программа мгновенно классифицирует новые файлы по типам и раскладывает их в папки, которые вы сами назначите. Устаревшие файлы автоматически удаляются по заданному расписанию.

Работает по принципу: **«установил и забыл»**.

---

## ✨ Ключевые возможности

### 🎯 Гибкая маршрутизация
Вы можете указать **любую папку для любого формата**:

| Формат | Целевая папка (пример) |
|--------|------------------------|
| `.pdf`, `.docx`, `.txt` | `~/Документы/Учёба` |
| `.jpg`, `.png`, `.gif` | `~/Изображения/Архив` |
| `.exe`, `.msi`, `.dmg` | `~/Дистрибутивы` |
| `.zip`, `.rar`, `.7z` | `~/Архивы/Входящие` |
| `.mp4`, `.avi`, `.mkv` | `~/Видео/Загрузки` |

### 🧹 Автоматическая очистка
Файлы, которые не трогали больше N дней (настраивается), автоматически отправляются в Корзину.

### 💻 Кроссплатформенность
Работает одинаково на **Windows**, **macOS** и **Linux**.

### ⚙️ Простая настройка
Через графический интерфейс или редактирование JSON-конфига.

### 🔔 Фоновый режим
Программа живёт в системном трее и не мешает работе.

### 📝 Журнал действий
Все перемещения и удаления записываются в лог. Вы всегда знаете, что произошло.

### ↩️ Отмена последнего действия
Случайно переместили не туда? Нажмите Undo — файл вернётся.
